How to Play:
Requires Java
Run either start.bat (Windows) or start.sh (Linux) to run the game.


Controls:
W, A, S, D and the mouse.
F1 to take screenshot
P - Pause
L to turn lights on/off
C - Toggle detach camera


Features:
Randomly generated maps.


Other Notes:
There is a config file (config.txt) in the root directory for tweaking the game.  It should be self-explanatory.
